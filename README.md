# CHIA blockchain

## Build

sh install.sh

```. ./activate
chia init
chia keys generate```



```. ./activate
chia plots create -k 32 -n 2
chia plots check -n 30 ```

## Launch

```. ./activate
sh install-timelord.sh
chia start timelord &```
